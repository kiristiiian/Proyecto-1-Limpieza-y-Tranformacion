# Proyecto-1-Limpieza-y-Tranformacion
Tratando con DataSet de Netflix (Limpieza y Transformación)

Demostrar capacidad de saber lidiar con “datos sucios”, es decir, datos que requieren cierto tipo de preparación previo a su análisis o utilización en una aplicación puntual.
Hacer el trabajo duro necesario para limpiar un dataset y dejarlo en condiciones de ser analizado.

Limpieza de datos: 
Eliminar valores nulos.
Reemplazar valores incompletos (e.g. reemplazar con promedios)
Transformación: 
Convertir columnas a un tipo de datos específico
Crear métricas o features a partir de determinadas columnas.

DATASET Netflix DATA

Preguntas de guía
¿Cuántos valores nulos encontrás en los datos? ¿Los puedes eliminar?
¿Cuántos valores incompletos encontrás en los datos? ¿Los puedes reemplazar?
¿Podés eliminar columnas que no te aportan información? ¿Cuáles son? ¿Por qué las eliminarías?
¿Qué tipo de dato es la columna “release_year”? ¿Lo podes convertir a integer?
La columna “listed_in” contiene diferentes valores separados por coma, ¿Podés crear una columna y quedarte con el primer valor?
